Tairu CLI

Commands​

tairu open​

Open and display a Tile by URL.

Usage: tairu open <URL>

tairu serve​

Serve a local Tile handler.

Usage: tairu serve <PATH> [--port <value>]

Flags​

  • --port=<value>: Server port (default: 3210)
