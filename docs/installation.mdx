# Installation

## Requirements

[Node.js](https://nodejs.org/en) v20 or higher and a package manager such as npm are required to install and use the Tairu framework.

## Tiles creations

The handler package is needed to create Tiles server-side.

<Tabs groupId="package-manager">
  <TabItem value="npm" label="npm" default>
    
    ```sh
    npm install @tairu/handler
    ```

  </TabItem>
  <TabItem value="pnpm">
  
    ```sh
    pnpm add @tairu/handler
    ```
  
  </TabItem>
</Tabs>
